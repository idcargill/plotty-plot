Thanks for visiting Point Plotter!

This site was built for entering single point information or uploading a data set you might get form an api endpoint. Personally I use it for checking wilderness backcountry point references. You can use it for whatever you like!

Plotty Plot was created as a testing ground and learning project for front end development techniques. It was build using basic HTML, CSS, and JavaScript with 3rd party mapping APIs.

About me:

I am an aspiring software developer. I like to dabble in the interaction of UI and data and think API end points are the bees knees.
