JSON FILE

File drop takes a .json file of key:value pairs.
There must be one key of 'lat' or 'latitude' and another of 'lon' or 'longitude'.
Case and location of the keys does not matter.

{data: 'Details...', lat: 50.2, Longitude: -122.24, ...}

Individual Points

Add points using Degrees/minutes/seconds or Coordinates.
All points will be added to the same layer with any uploaded data.
